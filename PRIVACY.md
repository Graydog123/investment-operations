# 隐私与敏感信息规则

本项目公开思想、方法和可核验案例，不公开任何人的真实投资账户。

## 禁止提交

- 持仓明细、成交记录、盈亏和资金余额
- 券商名称与账户号码、融资额度、担保比例截图
- 身份证件、手机号、邮箱、地址和设备信息
- 私人聊天记录、未授权录音和内部文件
- 能够通过组合推断出个人财富、交易身份或工作单位的信息

## 举例方式

需要说明仓位、融资或回撤时，请使用：

- 明确标注为“虚构示例”的数据；或
- 归一化数据，例如把净资产统一设为100；或
- 已公开且有可靠来源的历史案例。

归一化示例也不得从真实个人账户逐笔转换而来。

## 发现隐私问题

如果你发现仓库中存在可能泄露个人信息的内容，请不要在公开 Issue 中复述敏感数据。可在仓库启用后通过 GitHub 的私密安全报告渠道联系维护者。

维护者会优先移除公开内容；但 Git 历史和第三方镜像可能保留旧版本，因此发布前的预防审查比发布后的删除更重要。
