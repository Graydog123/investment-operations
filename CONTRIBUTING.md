# 贡献指南

感谢你愿意帮助完善《投资经营学》。本项目面向普通投资者，最重要的标准是：**通俗易懂，同时不牺牲事实准确性和使用边界。**

## 欢迎的贡献

- 修正错别字、语病、断链和格式问题
- 补充公司公告、年报、监管文件和学术论文等原始来源
- 提供能够挑战现有命题的反例
- 把金融黑话改写为更有生活画面、但仍准确的表达
- 补全模型的适用条件、失效条件和行动约束
- 改进公开文档转换、校验和发布工具

## 不接受的内容

- 荐股、喊单、目标价或收益承诺
- 未经核实的客户、订单、供应链和“唯一供应商”说法
- 把送样、验证或布局直接写成量产和收入
- 只有单日涨跌、没有证据链的市场结论
- 真实账户、持仓、交易、盈亏、券商截图或个人身份资料
- 无权公开的付费报告、私人文档或大段受版权保护内容

## 内容提交标准

1. 明确区分事实、判断和未验证信息。
2. 优先引用原始资料，并提供可访问链接。
3. 一个案例不能自动证明普遍规律；同时说明反证和边界。
4. 专业术语尽量遵循“生活 → 经营 → 金融 → 投资”的翻译路径。
5. 新术语应比现有术语更准确、更易用，而不是为了原创而原创。
6. 需要数字举例时，使用公开数据或明确标注的虚构、标准化数据。

## 提交流程

1. 先搜索现有 Issue，避免重复。
2. 小修改可以直接提交 Pull Request；较大改动请先开 Issue 说明目的和边界。
3. Pull Request 中说明：改了什么、依据是什么、如何验证、可能影响哪些章节。
4. 内容贡献默认按 CC BY-NC-SA 4.0 许可；`tools/` 下代码贡献按 MIT License 许可。

## 写作检查

提交前可以用四个问题自查：

- 普通读者能在脑海里看到画面吗？
- 一个开了二十年小饭馆的老板能听懂吗？
- 这段话能指导行动，而不只是听起来正确吗？
- 它说明了什么时候不再适用吗？
