## 推荐路径

- 想连续阅读第一部：序章 → 第一章 → 第二章 → 第三章 → 第四章
- 想连续阅读第二部：第五章 → 第六章 → 第七章 → 第八章
- 想连续阅读第三部：第九章 → 第十章 → 第十一章 → 第十二章 → 第十三章 → 第十四章
- 想从市场进入公司研究：第五章 → 第六章 → 第七章 → 第八章 → 第九章
- 想快速理解账户经营：序章 → 第四章 → 第九章 → 第十四章 → 模型图谱
- 想理解专业概念：投资翻译词典 → 第一性命题
- 想检查证据：案例验证库 → 对应模型和样章
- 想参与共建：先读 [贡献指南](../CONTRIBUTING.md)
- 想理解正文怎样重构：三章横向校对报告 → 正文重构模板

> 本项目不提供荐股、实时交易信号或收益承诺。公开案例只用于方法研究，不代表当前投资观点。
