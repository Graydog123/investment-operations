# 思想体系总图

四册不是四套平行内容，而是一条从理解到行动的翻译链。每往后一层，都必须能回到前一层解释，也必须能被后一层检验。

> 总装原则<br>术语负责让人看见，专业翻译负责准确，命题负责判断，模型负责行动，案例负责证伪，主书负责讲清楚。

# 四册各自只做一件事

联校的第一步，是阻止每一册越权。词典不能替代专业定义，命题不能冒充模型，模型也不能冒充自动交易系统。

# 跨册边界审计

联校最重要的产出不是发现多少相似词，而是决定相似词各自只负责什么。以下边界将成为后续正文的统一口径。

> 经营权—等待权—退出权<br>经营权是账户继续自主配置资本的总资格；等待权解决时间，退出权解决流动性，未承诺资本提供物质基础。<br>误用边界　不要把四个词并列成同义口号。

> 客流—翻台<br>客流是当下真实购买力量，翻台是筹码交换速度。价格方向、成交活跃与后续承接必须共同判断。<br>误用边界　成交放大不自动等于新增客流。

> 翻台—总资产周转率<br>前者属于市场交易语言，后者属于公司经营效率。<br>误用边界　正文中分别使用‘市场翻台’和‘资产周转’，避免一个词跨两个层次。

> 烂库存—库存周期<br>烂库存指账户持仓理由变质；库存周期指企业或行业实物库存变化。<br>误用边界　二者可以相互影响，但不能互换。

# 跨册边界审计（续）

> 错误预算—止损—最大回撤<br>错误预算是事前允许的账户损失；止损是执行工具；最大回撤是结果统计。<br>误用边界　不能用一个固定止损百分比替代错误预算。

> 核心店—护城河<br>核心店是当期产业和资金地位；护城河是长期商业竞争优势。<br>误用边界　市场核心可能迁移，好公司也可能暂时不是交易核心。

> 证据升级—催化剂<br>证据升级描述事实成熟度；催化剂描述市场重新定价的时间触发。<br>误用边界　有催化不代表证据成熟，证据成熟也不保证立即上涨。

> 隐含菜单—估值指标<br>隐含菜单从价格反推未来要求；P/E、P/B、PEG和DCF只是使用的尺子。<br>误用边界　不要把模型退化成倍数高低判断。

# 跨册边界审计（续）

> 收银台链—自由现金流<br>收银台链是从行业需求到股东现金的完整路径；自由现金流是链条后端的重要结果。<br>误用边界　企业自由现金流与账户现金、未承诺资本必须分开。

> 固定租金—经营杠杆—财务杠杆<br>固定租金在账户语境主要指确定资金义务；经营杠杆属于公司的固定经营成本，财务杠杆属于债务结构。<br>误用边界　出现‘房租’隐喻时必须注明讨论的是公司还是账户。

> 街区迁移—行业周期<br>街区迁移描述资金与市场偏好的持续搬迁；行业周期描述供需和利润的现实变化。<br>误用边界　单日轮动不是街区迁移，资金迁移也不自动证明产业周期反转。

> 复盘制度化—诚实记录<br>复盘制度化解决怎么复盘，诚实记录解决是否允许不利事实进入系统。<br>误用边界　流程完整不等于结论诚实。

## 统一写作规则

先说分析对象　每次出现库存、现金、杠杆和周转，先说明讨论的是公司、行业、市场还是账户。

隐喻后接专业名　第一次出现原创术语时，紧跟专业映射；后文可只用原创语言。

命题必须能下行动　没有观察变量、适用条件和停止条件的句子，不进入一级命题。

模型不得自动交易　模型输出的是观察次序和动作约束，不输出脱离账户的固定买卖比例。

案例必须允许反证　每个正面案例同时寻找反例和误用边界，禁止只挑成功故事。

# 下一册接口：案例验证库

> 下一步不是讲更多故事<br>每个案例都要承担检验任务：它支持哪个模型、反驳哪个误用、缺少什么证据，以及换一个行业或账户后是否仍然成立。

| 材料 | 必须回答 | 用途 |
| --- | --- | --- |
| 正例 | 模型在什么条件下有效 | 确认机制而非崇拜结果 |
| 反例 | 相似外观为何得到相反结果 | 发现隐藏变量和适用边界 |
| 误用例 | 正确模型怎样被用错 | 识别跳级、偷换对象和事后解释 |
| 散户场景 | 普通账户怎样观察和行动 | 把历史故事翻译成可执行约束 |
| 原始证据 | 公告、财报、监管文件和一手资料 | 区分事实、判断与未验证 |

## 完成标准

1. 每个一级模型至少有一个正例、一个反例和一个散户账户场景。

2. 案例事实可追溯到原始资料；推断与事实分开标注。

3. 案例能够修正模型，不能只是给既有观点配插图。

4. 完成后把30个模型筛成10—15个一级核心，其余降为子模型。

5. 只有模型v0.2稳定后，才回到主书进行正文重构与压缩。

总装之后，体系才第一次具备可检验性。

生活语言让人看见，专业语言防止说错；
命题决定相信什么，模型决定怎样行动；
案例最终决定，这套思想能不能留下。
