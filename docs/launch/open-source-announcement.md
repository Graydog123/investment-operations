# 把金融还原为生活：我把《投资经营学》开源了

我把《投资经营学》开源了。

做这件事的出发点很简单：希望它能为真正需要帮助的普通散户，提供一套更容易理解、也更能保护账户的投资语言。

很多散户并不缺少努力。他们看新闻、读研报、研究公司，也愿意学习各种指标。真正困难的地方，往往是专业金融语言与日常判断之间隔着一道墙。概念记住了，到了市场剧烈波动、持仓亏损或者热点快速轮动的时候，却不知道应该怎样行动。

《投资经营学》想做的，是把这道墙拆掉。

我们不从术语开始，而从普通人已经拥有的生活经验开始：

- 股价像客流，成交量像翻台；
- 持仓像库存，放久了不会自动升值，也可能失去原来的经营理由；
- 融资像房租，它增加的不只是收益弹性，也会缩短等待时间；
- 主线像街区，繁荣会吸引顾客，也会吸引更多竞争者；
- 现金不是无所作为，而是尚未承诺出去的未来；
- 退出不是给过去定罪，而是把未来重新交给自己。

这套方法遵循一条固定路径：

> 生活画面 → 经营含义 → 金融概念 → 投资行动

它不是要把金融知识讲得肤浅，而是希望先让人真正理解，再告诉他专业名称。一个开了二十年小饭馆的老板，即使没有读过金融，也天然懂得客流、库存、房租、现金和关店。如果这些经营直觉能够被准确地带回投资，复杂概念就不再只是黑话。

## 目前开放了什么

首个公开版本包括：

- 《投资经营学词典》：定义等待权、经营权、菜单失效、街区迁移等核心语言；
- 《投资翻译词典》：把金融概念翻译成生活与经营经验；
- 《第一性命题》：保留可验证、可行动、可复述的判断；
- 《模型图谱》：说明概念之间如何连接，以及什么时候停止使用；
- 《案例验证库》：用公开案例和反例检验模型；
- 序章与第四章样章。

这些内容还不是终稿。开源不是宣布一套理论已经完成，而是让它更早地接受现实、来源、反例和不同投资周期的检验。

## 它不会做什么

这个项目不荐股，不提供实时买卖信号，不预测目标价，也不承诺收益。

我也不会公开任何人的真实持仓、账户截图、成交记录或财富信息。案例的作用是检验方法，不是展示战绩，更不是诱导别人复制交易。

如果一个方法只能靠成功故事证明自己，不能说明何时失效，它还不是一套可以依赖的系统。

## 为什么选择开源

投资方法最容易犯的错误，是只收集支持自己的证据。把项目公开，意味着任何人都可以指出事实错误、补充原始来源、提交反例，或者告诉我某个概念虽然漂亮，却无法指导行动。

我希望它最终不是一本只能被阅读的书，而是一套能够长期维护的公共思想资产：术语可以被澄清，命题可以被降级，模型可以被反例击中，历史也不会在结果出来以后被悄悄改写。

如果这套语言能够帮助一个普通投资者少追一次高潮、少扛一次已经失效的仓位、少在融资压力下失去等待权，或者更早意识到账户生存比证明自己正确更重要，它就已经产生了价值。
