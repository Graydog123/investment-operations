# 第十四章　先保住店

风险管理不是让账户永远不跌，而是避免一次错误永久失去经营资格。

## 两家店遇到同一个淡季

同一条街上，两家店的客流都下降三成。

第一家没有贷款，冰箱里只备短期货，老板留着几个月现金。他可以缩短营业时间、换菜单、等租金重新谈判，也可以趁隔壁转让时接下一间更好的铺面。

第二家刚借钱扩店，长期货压满仓库，房租和工资都在月底到期。客流下降的幅度一样，它却没有时间等待。为了付账，只能在最差的时候打折清货、裁人、关店。

外部冲击相同，命运不同。区别不只在资产跌了多少，而在下跌以后还剩多少选择。

> 真正的风险不是价格下跌，而是下跌后你失去选择。

## 波动、永久损失与强制退出

“风险”常被混成一个词，至少应拆成三层。

### 普通波动

价格在正常范围内上下变化，买入理由、资金期限和账户结构没有失效。它令人不舒服，但不必然要求行动。

### 永久性损失

公司竞争力、现金流、治理或产业地位发生难以恢复的破坏，原本期待的价值不再存在。没有融资也会受伤。

### 强制退出

资产可能仍有价值，账户却因保证金、生活支出、赎回、低流动性或心理崩溃，被迫在不利位置卖出。它把可能暂时的损失固定下来，并取消后续参与。

风险管理不能只看波动率。真正要防的是后两层，尤其是资产判断尚未尘埃落定，账户已经先失去选择。

## 硅谷银行：资产还在，时间先消失

美联储2023年对硅谷银行失败的复盘指出，银行快速增长，但治理与风险管理没有同步；流动性风险、利率风险和投资组合管理存在严重缺陷。报告还讨论了其高度集中的存款基础、大量未保险存款以及快速流出。[1]

美联储监察长办公室后续的重大损失复盘进一步总结：银行客户集中在科技和生命科学领域，未保险存款比例高，现金流大且不规则；银行又把大量存款投入期限较长的证券，利率上升后出现显著未实现损失。报告称，沟通和外部事件共同触发了巨额存款外流请求。[2]

银行挤兑与个人股票账户不是一回事。银行有存款负债、监管资本和支付义务，资金流出速度也远高于普通投资者。本章只借用一个机制：账面上有资产，不等于拥有时间；资金来源集中、期限错配和流动性不足，会在压力中迅速夺走选择。

对散户而言，对应问题不是“会不会被挤兑”，而是：谁有权要求你何时卖？融资方、生活账单、即将使用的资金，还是自己无法承受的心理压力？

## 风险红线要按顺序设置

账户生存不是从单股止损开始。更合理的次序是：

### 第一层：生活资金与投资资金分离

未来一段时间明确要使用的钱，不承担需要长期等待的风险。房租、医疗、教育和家庭应急资金，不因为“机会难得”临时进入市场。

### 第二层：先管理负债

融资会把价格波动变成合同问题。主线越拥挤、波动越大、退出越困难，越不应提高借款。大赚以后，优先降低融资和固定承诺，而不是把全部利润再次投入同一方向。

### 第三层：管理同一主线总暴露

五只股票可能只是一种错误。压力测试要按共同失效原因合并，而不是逐股看上限。

### 第四层：管理单只股票

公司事件、治理和流动性风险不能被行业分散掩盖。即使主线仍强，个股也可能永久失效。

### 第五层：最后才讨论等待价值

只有前四层都允许等待，长期价值才有机会兑现。用短期资金、融资和高集中持有“长期好公司”，只是给期限错配起了一个好听名字。

## 长期投资首先要求长期资金

长期逻辑不能延长债务到期日，也不能暂停家庭支出。持有周期应同时满足三种时间：

- 公司兑现需要多久；
- 市场可能错多久；
- 你的资金能等多久。

前两项通常只能估计，第三项必须诚实。若资金在六个月后有明确用途，就不能把它配置给可能需要三年验证的逻辑；若融资条件可能在下跌时变化，就不能把债权人的时间表当作自己的长期耐心。

> 持有愿望不能延长资金期限。

## 心理资本也是资本

有些仓位数学上没有触发红线，却已经让人无法睡眠、无法阅读反方材料、不断看盘和冲动交易。此时账户仍有现金，决策能力却开始枯竭。

心理资本不能靠主观一句“我难受”来任意管理，否则任何波动都可能成为追涨杀跌的理由。它需要行为指标：

| 预警信号 | 可能说明 |
| --- | --- |
| 睡眠连续受影响 | 仓位或不确定性超过承载能力 |
| 看盘和下单频率突然升高 | 焦虑正在替代计划 |
| 无法记录反证 | 研究开始服务于持仓 |
| 频繁修改失效条件 | 账户在拖延承认变化 |
| 家庭现金被临时挪用 | 投资已侵入生活经营 |

当多个指标同时出现，先降到能恢复判断的规模，再研究公司。减仓不一定代表看空，它也可能是购买清醒。

> 让你无法执行规则的仓位，即使数学上没超限，也已经过大。

## 伯克希尔：流动性是防守，也是进攻资格

伯克希尔在2023年股东信中强调长期保持极端财政保守和充足流动性，并回顾2008年公司没有依赖商业票据、银行授信或债务市场满足现金需要。[3]

个人投资者不能复制伯克希尔的保险浮存金、多元业务和融资条件，也不应据此得出“现金越多越好”。现金有机会成本，长期过度防守也会损害复利。

该案例真正说明的是：流动性不仅用于避免死亡，也用于在别人被迫退出时仍能行动。生存和收益并非对立；保留经营权，本身就是未来收益的前提。

## 一张选择权压力测试

不要只测试“组合下跌10%会亏多少钱”，还要画出下跌后的行动集合。

| 情景 | 要检查的选择 |
| --- | --- |
| 市场整体下跌10% | 是否仍有现金；融资比例如何变化 |
| 同一主线下跌20% | 哪些持仓会同步失效；需要多久退出 |
| 最大持仓停牌或跳空 | 其他资产能否提供缓冲 |
| 三个月没有收入或出现家庭支出 | 是否需要从账户取钱 |
| 连续五笔交易失败 | 心理与规则是否仍能执行 |

压力测试的结论不是预测这些事情一定发生，而是判断如果发生，账户还有几条路。只有一条路——低位卖出——说明选择权已经过薄。

## 生存不等于永远防守

“先保住店”最容易被误用成永久低仓位、永远持有现金、拒绝一切集中。那样虽然减少波动，也可能失去长期回报。

经营良好的餐馆不会因为淡季可能出现就永远不开门。证据成熟、赔率充足、流动性良好、错误预算可控时，资本可以扩张；主线进入明确主升阶段，合理提高风险也可能是系统的一部分。

区别在于：扩张不能押上生活资金，不能让一次普通回撤触发被迫退出，也不能让单一判断决定全部未来。

## 店还开着，才有下一桌顾客

投资生涯不由一笔交易决定，也不由一轮牛市决定。复利需要资本、时间和判断能力持续存在。一次永久退出，会截断所有尚未出现的机会。

所以，账户的首要产出不只是收益率，还包括下一次配置资本的能力。生活资金分离、负债受控、同因子不过量、单股可承受、心理仍能执行——这些条件共同组成经营权。

风险管理不是让账户不受伤，而是让受伤之后仍有路可走。

> 生存不是收益的对立面，而是所有未来收益的前提。
