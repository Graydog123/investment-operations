# 第二十三章　最好的店，也要有退出方案

退出不是给过去定罪，而是停止让过去继续决定未来资本的去向。

## 关掉一家分店，不等于否定整个品牌

一家连锁餐馆有一间分店，最初选址合理，开业后也排过队。后来街区迁移、租金上涨、顾客减少，继续经营只会不断占用现金。

老板决定关店。有人说：“既然品牌没坏，为什么要关？”

因为品牌是否有价值，与这一间店是否仍值得占用资本，是两个问题。

股票也一样。卖出结束的是当前价格、当前仓位和当前条件下的一项资本承诺，不是宣布公司永远没有价值，更不是承认自己从一开始就愚蠢。

> 退出不是承认过去错误，而是重新把未来交给自己。

## 买入计划没有退出，只写了一半

不同交易身份，对应不同失效方式。

趋势交易买的是主线、相对强度和价格结构；价值投资买的是价格相对长期现金价值的折价；事件交易买的是事件发生概率、兑现时间和市场预期差。

如果买入时只写“为什么会涨”，没有写“什么变化说明这笔生意不再成立”，亏损以后就只能临时解释。最常见的结果是：趋势失效以后改谈长期空间，估值过高以后改谈公司质地，事件落空以后改谈下一次催化。

完整计划至少写两条：

- 资产失效：公司、价格、趋势或事件的哪项条件被破坏；
- 账户失效：仓位、融资、资金期限或流动性的哪项条件不再允许等待。

第二条常被忽略。公司可能没有失效，账户却已经没有继续持有的资格。

## 四类失效，不要用同一把尺子

### 基本面失效

决定性变量被证伪，例如产品无法通过关键验证、核心客户需求消失、竞争优势被替代、现金流或治理发生结构性破坏。

它不等于一季利润低于预期，也不能凭传闻直接确认。需要原始来源和证据等级。

### 估值失效

公司仍然优秀，但价格要求的增长、利润率、持续时间或资本回报已远高于合理证据。此时卖出不是看坏公司，而是拒绝继续支付更昂贵的剧本。

### 趋势失效

主线退潮、相对强度持续下降、关键结构破坏，或者板块修复时个股仍不能获得承接。对于短线和趋势身份，市场证据失效就足以改变仓位，不必等待长期基本面证伪。

### 账户失效

融资、同因子暴露、生活资金需求、流动性或心理资本越过红线。资产可能仍然值得研究，账户也必须先恢复经营权。

四类失效可以同时发生，也可能互相独立。退出之前先标注类型，才能决定是一次性结束、分层减仓，还是只降低规模。

## 星巴克：关店可以是修复经营的一部分

2008年6月，星巴克在提交给美国证券交易委员会的文件中披露，计划关闭约600家表现不佳的美国直营店；公司称该决定来自对美国门店组合的严格评估。[1]

关店涉及宏观环境、扩张质量、门店经济和企业转型等多重背景，不能简化成“关店以后公司一定复苏”。它也不是个人投资者机械止损的证据。

这个案例只说明一条经营原则：优秀品牌也不必保留每一项旧承诺。关闭低效门店，可以释放管理注意力、租约、人员和现金，让企业重新配置资源。资产退出与企业价值并不矛盾。

## 英特尔：曾经的招牌，也可能不再是未来的厨房

英特尔的官方历史资料记载，公司在1985年面对半导体行业压力时停止生产DRAM，并把经营重心转向未来方向；英特尔同时保存了其早期以存储产品起家的历史资料。[2]

企业战略退出与个人股票卖出并不相同。企业拥有员工、工厂、客户和长期技术路线，转换成本远高于证券账户。这个案例也不能用来证明英特尔后来所有战略都正确。

它提供的只是更深一层提醒：退出不只属于失败资产。某项业务曾经是招牌、曾经贡献利润、甚至塑造公司身份，也不能获得永久豁免。资本要回答的始终是未来，而不是历史荣誉。

## 退出可以分层，不必只有清仓与死扛

经营者常把退出理解成二选一：全部卖掉，或完全不动。更实用的方法是让动作与失效程度匹配。

| 层级 | 典型状态 | 动作 |
| --- | --- | --- |
| 停止扩张 | 新证据不足、赔率恶化 | 不再加仓，取消计划订单 |
| 角色降级 | 从核心变成普通跟随 | 降低仓位和观察期限 |
| 释放重复暴露 | 同一主线或同一风险因子过量 | 先减弱者和重复者 |
| 主体退出 | 原交易身份已明确失效 | 收回大部分或全部资本 |
| 账户急停 | 融资、流动性或生活资金触线 | 优先恢复经营权，不等完美价格 |

分层不是为了拖延。每一层都要有完成时间和下一层触发条件；否则“先减一点”会变成没有终点的心理安慰。

## 先减弱者，再减核心；核心没有永久豁免

组合调整通常从失去相对优势、证据停滞和重复暴露的仓位开始。因为这些资产正在消耗错误预算，却没有提供相应的主线地位或赔率。

但“先减弱者”不是保护核心的法律。如果公司事件直接证伪核心，或者主线的核心本身已经失去承接，就不能机械等待后排先卖。核心仓位最大，往往也是账户失效的主要来源。

判断顺序可以是：

1. 哪项风险正在缩短账户时间；
2. 哪只持仓已失去原交易身份；
3. 哪些股票只是同一种错误的重复代码；
4. 主线核心是否仍在确认整个方向；
5. 资本收回以后去哪里。

减仓不是追求持仓平均，而是让未来损失重新回到账户可以承担的范围。

## 资本去向决定退出质量

卖出以后，资本至少有四种去向：

- 偿还融资，恢复等待权；
- 变成现金，保留未承诺资本；
- 转向证据和赔率更好的资产；
- 留作明确的生活或经营用途。

如果卖出只是为了缓解难受，随后立刻追入另一只高潮股票，账户并没有获得更多选择，只是更换了库存标签。

所以，退出计划必须写“卖什么”，也要写“卖出以后做什么”。资本去向无法说明时，可以先降低动作级别，重新诊断；账户红线已经触发时，去向则优先是现金和偿债。

## 重新买入不是打脸，而是新交易

退出机制如果没有重返条件，投资者会陷入另一种自尊：卖出以后不愿再看，生怕重新买入等于承认此前卖错。

市场和公司都会变化。原失效条件可能被新事实解除，主线可能重新形成，价格可能重新提供赔率。重新进入并不推翻旧决定，因为信息集、价格和账户状态已经不同。

退出时同时写下三类重返条件：

- 证据：什么新事实恢复原逻辑；
- 市场：主线、相对强度或价格结构如何重新确认；
- 账户：错误预算、现金和相关暴露是否允许重新承诺。

> 卖出以后可以重新开门，但必须重新签租约。

重新买入不能只因为股价上涨、自己后悔或担心卖飞。它必须像任何新交易一样重新通过证据、赔率、仓位和退出检查。

## 本章行动卡

### 做什么

- 买入前同时写资产失效和账户失效条件。
- 明确失效属于基本面、估值、趋势还是账户。
- 让退出动作与失效程度匹配，并写明完成时间。
- 写清资本去向和重新进入条件。

### 不做什么

- 不用长期公司逻辑替短线趋势交易续命。
- 不把“核心”理解为永久豁免。
- 不为缓解情绪卖出后立刻追入另一只高潮股。
- 不把卖出公司永久拉黑，也不因卖飞立即追回。

### 适用条件

适用于趋势、价值、事件和组合再平衡。长期指数投资可以用资金用途、资产配置和再平衡条件代替个股失效规则。

### 失效条件

战略持股、税费、锁定期、停牌和低流动性会改变退出节奏；但这些现实约束只会提高事前设计的重要性，不会让退出问题消失。

### 主要风险

一是把退出做成机械价格止损，忽略波动和交易身份；二是把分层退出做成拖延；三是重新进入时只因后悔而提高价格要求、降低证据要求。
