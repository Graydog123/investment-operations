# 第十九章　利润不是营业额

收入属于公司，只有穿过成本、投资与资本配置后，现金才可能属于股东。

## 门口排长队，老板口袋里未必有钱

一家餐馆每天营业额十万元，看起来生意兴隆。

可是顾客大量赊账，冰箱压满食材，新店正在装修，设备需要更换，银行贷款也要偿还。月底关门以后，老板发现收银台没有现金，还要继续借钱维持扩张。

营业额是真的，忙碌也是真的。股东变富却未必是真的。

> 收入属于公司，现金才开始属于股东。

财报研究不能停在“收入增长”和“净利润增长”。它要继续追问：钱有没有回来、为了增长又投了多少、最后由谁决定这笔钱去哪。

## 从顾客付款到股东口袋

一元收入要经过多道门。

| 层次 | 餐馆语言 | 投资要问什么 |
| --- | --- | --- |
| 收入 | 顾客点了多少菜 | 需求是真实还是一次性 |
| 毛利 | 扣除食材后剩多少 | 产品有没有定价权 |
| 经营利润 | 再扣工资、房租和运营费用 | 生意本身是否赚钱 |
| 经营现金流 | 账上的利润是否收成现金 | 应收、库存和付款节奏怎样 |
| 自由现金流 | 再扣维持与扩张厨房的钱 | 真正可配置的现金有多少 |
| 资本配置 | 现金用于扩店、还债还是返还 | 每一元钱去向是否创造价值 |
| 每股价值 | 老板最终分到多少 | 是否被增发、稀释和高价并购吞掉 |

任何一层都可能截断上一层的好消息。

收入增长但毛利下降，可能是靠降价换量；利润增长但现金不回，可能是赊账增加；经营现金流很好但资本开支巨大，自由现金流仍可能很少；公司现金很多但高价并购和无效扩张，股东仍未受益。

## 利润和现金为什么会分开

会计利润按照权责发生制确认，不必等到现金真正收回。

常见差异包括：

### 应收账款

货已经卖出、收入已经确认，客户还没付款。适度信用销售是正常经营；应收持续快于收入增长，则要检查客户质量、回款和是否通过放宽账期换收入。

### 库存

公司先把现金变成原料和产品。需求强劲时，备货支持增长；需求转弱时，库存会占用现金并面临减值。

### 预付款与合同资产

项目型业务可能先投入、后验收。利润与现金的时间差可以合理，但周期越来越长、争议越来越多时，风险也会增加。

### 非现金项目

折旧、减值、股权激励和公允价值变化会使利润与现金出现差异。它们不能被统一称为“没有影响”。折旧虽非当期付款，却可能对应未来真实设备更新；股权激励不流出现金，却会稀释股东。

> 利润表写赚了多少，现金流回答这钱是否真的回来。

现金流也能被短期美化，例如延迟支付供应商、压低库存或提前收款。因此不能只看一个季度，应观察多个周期的方向和原因。

## 自由现金流不是“经营现金流减一个数字”那么简单

自由现金流通常从经营现金流中扣除资本开支，但资本开支至少分两类：

- 维持性投入：不花就会让现有生意退化；
- 扩张性投入：为了新增产能、门店和未来增长。

财报不会总是替投资者清楚分开。若把所有资本开支都当成扩张，可能高估现有业务现金能力；若把所有投入都当成维持，又会低估高回报增长。

更实用的办法是长期观察：新增资本带来了多少新增利润和现金？设备、门店和研发投入之后，单位经济是否改善？

增长如果不能提高资本回报，只是在更大规模上重复忙碌。

## 安然：报表可以改变名字，不能改变经济实质

美国证券交易委员会2005年关于表外安排的报告讨论了安然部分“预付”交易。报告指出，相关交易的经济实质类似借款，但安然没有把收到的现金列为融资现金流，也没有把未来付款义务作为债务列示，而是形成了经营现金流的假象并隐藏债务。[1]

安然涉及欺诈和复杂结构，不能把它类比为所有利润与现金背离，也不能因为财报复杂就推定公司有问题。

它提供的极端边界是：投资者不能只接受报表科目名称，还要还原整组交易的经济实质。现金从哪里来？未来需要怎样归还？风险和收益最终留在谁身上？

> 科目可以改名，现金的来源和去向不会因此改变。

## 会赚钱只是第一层，会放钱才决定复利

公司产生自由现金后，管理层通常有六种选择：

1. 投回原业务；
2. 开辟新业务或并购；
3. 偿还债务；
4. 分红；
5. 回购股票；
6. 保留现金。

没有一种永远正确。

原业务增量回报高，继续投资可能创造复利；机会不足时，偿债和分红可能更诚实；股价明显低于内在价值时回购有利，每股价值很高时回购则可能浪费现金；并购可以获得能力，也可以把好业务现金投入坏生意。

评价管理层，不能只听“我们重视股东回报”，要建立多年资本去向表：花了多少钱，去了哪里，事后产生多少增量现金和每股价值。

> 会赚钱只是第一层，会把赚来的钱放对地方才决定复利。

## 总额增长可能掩盖每股停滞

公司收入、利润和现金总额增长，股东每一股对应的价值仍可能不增长。

原因包括：

- 大量增发股票；
- 股权激励持续稀释；
- 高价并购带来商誉和新股；
- 债务上升把更多现金交给债权人；
- 回购只抵消员工激励，没有真正减少股本。

散户最终持有的是每股权益，不是公司新闻里的总规模。

所以，链条的最后一步必须是每股：每股自由现金、每股净资产、每股分红或每股内在价值是否提高。

## 一张六层现金审计表

研究一家公司时，可以按以下顺序：

| 检查层 | 主要问题 | 预警信号 |
| --- | --- | --- |
| 收入 | 增长来自量、价还是并购 | 一次性、低质量客户、放宽账期 |
| 利润 | 毛利与费用是否改善 | 降价换量、非经常收益 |
| 经营现金 | 利润是否收回 | 应收、库存快于收入 |
| 自由现金 | 增长消耗多少资本 | 资本开支长期吞噬现金 |
| 资本配置 | 现金去了哪里 | 高价并购、低回报扩张 |
| 每股结果 | 股东实际得到什么 | 稀释、负债、回购无效 |

单期异常不必立即判死刑。项目交付、季节性备货和高速增长都可能制造时间差。关键是原因是否透明、是否可逆、是否在随后周期兑现。

## 本章行动卡

### 做什么

- 沿收入、利润、经营现金流、自由现金流、资本配置和每股价值逐层核验；
- 利润增长时同步检查应收、库存、合同资产和付款节奏；
- 区分维持性投入与扩张性投入，并观察新增资本回报；
- 建立多年资本去向表，检查并购、偿债、分红、回购和现金；
- 最终用每股结果而不是公司总规模评价股东回报。

### 不做什么

- 不把营业额、净利润或经营现金流中的任一项当作终点；
- 不把所有非现金费用都视为无关；
- 不用单季度现金流美化下长期结论；
- 不因报表复杂就推定欺诈，也不因科目合规就停止追问经济实质；
- 不把增发后的总利润增长等同于每股增长。

### 适用条件

适用于制造业、消费、平台、并购型公司和资本密集型成长企业的财务质量与股东回报研究。

### 失效条件

金融企业、早期研发公司和重大并购重组期现金结构特殊，需要行业专属口径；非经常项目或会计政策变更主导当期数据时，应暂停单期结论并重构多期口径。

### 主要风险

自由现金流估算也包含判断，尤其是维持性与扩张性资本开支的划分。最危险的不是估算不精确，而是只选对持仓有利的口径。所有调整都应可重复、可对照、可解释。
