# 第十一章　潮水褪去以后

退潮不是突然创造风险，而是停止奖励繁荣期早已存在的错误。

## 旺季会把坏店也坐满

一条街突然成了热门景点。周末从早到晚排队，位置一般的店坐满，菜单混乱的店也坐满，成本高、服务差、靠借钱扩张的店同样坐满。

几个月后，老板们都觉得自己找到了成功公式。有人提高房租承受上限，有人把三家店同时扩成十家，还有人把临时爆款当成永久招牌。因为每一次冒险都赚了钱，原先的谨慎看起来像胆小，原先的红线看起来像妨碍增长。

客流下降时，问题仿佛一夜之间出现：空桌、库存、房租、工资和贷款同时压来。其实它们早已存在，只是旺季收入一直替它们付账。

> 繁荣最危险的作用，是让错误长期得到奖励。

## 盈利会给风险发一张能力证书

在单向上涨的市场里，许多不同策略会得到相似结果：研究扎实的核心持仓会上涨，高杠杆的同质化组合也会上涨；遵守规则的人赚钱，偶尔越过规则的人可能赚得更多。

结果相似，结构却完全不同。前者的收益可能来自证据、赔率和执行，后者可能主要来自环境、杠杆与运气。只看账户曲线，两者在旺季很难区分。

危险从第二步开始：人会把环境贡献归为自身能力，再用盈利证明可以承担更大风险。融资提高一点没有出事，下一次再提高；单股上限越过一次仍然盈利，例外便开始常态化；同一主线的股票越买越多，因为每一个代码都在证明方向正确。

风险标准通常不是在某一天被取消的，而是在多次“没出事”中逐渐漂移。每一次违规后的盈利，都会让下一次违规更容易。

> 风险标准不是一次放松的，而是在多次没出事中漂移的。

## 雷曼：退潮让多个承诺同时到期

美国金融危机调查委员会的最终报告讨论了雷曼兄弟在危机前的高杠杆、房地产相关敞口、短期融资依赖和透明度问题。[1] 雷曼并不是因为某一天突然学会冒险才失败；压力来临时，长期积累的资产、负债和融资结构同时被市场重新定价。

这个案例不能被简化成“杠杆必然导致破产”。金融危机由住房、证券化、评级、监管、融资市场和机构治理等多重因素共同造成。雷曼与个人股票账户在制度、资产和融资方式上也完全不同。

它对散户真正有用的部分只有一条：顺风时彼此独立的风险，会在压力中互相连接。资产下跌减少净值，融资方提高要求，市场流动性变差，卖出又加重价格压力。原本可以分开处理的问题，开始同时索债。

所以退潮复盘不能只问“哪只股票选错了”。还要问：为什么几只股票一起跌？为什么当时没有现金？为什么杠杆在价格下跌后反而升高？为什么原计划可以等待，实际账户却等不起？

## 回撤是一张显影纸

同样下跌10%，对不同账户不是同一件事。

无融资、持仓分散、资金期限长的账户，可能只经历净值波动；高融资、同一主线集中、生活资金混用的账户，则可能同时面对保证金、现金需要和情绪失控。市场给出相同价格变化，账户结构决定它会不会变成强制行动。

因此，回撤应拆成六本账：

> 回撤不是只在减少资产，也在暴露账户结构。

| 账目 | 要回答的问题 |
| --- | --- |
| 市场账 | 大盘和主要风格贡献了多少？ |
| 主线账 | 所属方向是正常分歧、阶段退潮还是趋势反转？ |
| 选股账 | 个股相对同类核心是否失去地位？ |
| 仓位账 | 单股与同因子规模是否放大损失？ |
| 负债账 | 融资和资金期限是否迫使行动？ |
| 执行账 | 是否偏离了事前计划，或在压力中反复切换？ |

这张表不是为了把亏损分摊得看起来合理，而是找出哪些损失来自环境、哪些来自账户自己制造的脆弱。

## 不要把外部冲击全部内化

退潮会显影结构，但不能把所有亏损都归咎于自己。政策突变、地缘冲突、系统性流动性收缩和无法预见的公司事件，都可能让合理策略遭遇损失。

如果把每次外部冲击都解释为“我本来应该知道”，复盘会制造事后全知。投资者可能由此走向另一个极端：永远轻仓、永远等待、再也不承担必要风险。

更准确的做法是分三栏：

- 可预见且可控制：例如融资过高、同因子过度集中、没有失效条件。
- 不可预见但可承受：例如突发事件造成跳空，但仓位使账户仍能行动。
- 不可预见且不可承受：事件无法预测，却暴露出规模和负债本来就不允许这种意外发生。

风险管理不是预测每次事故，而是让不可预测的事故不至于永久结束经营。

## 旺季也要做“无收益审计”

真正有效的退潮复盘，应该提前到上涨期。账户大赚时，暂时把收益数字遮住，只看结构：

1. 如果主线明天普遍下跌15%，净资产会损失多少？
2. 哪些持仓看似不同，却由同一批资金定价？
3. 融资会如何随净值下降而被动上升？
4. 压力日需要几个交易日才能退出？
5. 哪条规则最近被破例，破例只是因为它恰好赚了钱吗？

这叫无收益审计。它不否认趋势，也不要求在主升期机械降仓；它只是把“赚钱”从风险豁免证上拿下来。

伯克希尔在2023年股东信中再次强调极端财政保守与充足流动性，并回顾2008年公司不依赖商业票据、银行授信或债务市场满足现金需要。[2] 个人账户不能复制其保险业务、浮存金和现金规模，但可以理解其经营次序：流动性不是等危机发生后才临时寻找的。

## 退潮后的四种错误动作

第一种是把所有损失都归于市场。这样做保护了自尊，却让账户结构原封不动。

第二种是把所有损失都归于自己。这样做看似深刻，最后常常变成永久保守。

第三种是急着在最低点证明自己能扳回来。融资、频繁换股和追逐反弹，会让旧错误在新方向上继续扩张。

第四种是只记住痛苦，不改变制度。过一段时间行情回暖，同样的集中、杠杆和例外又会回来。

退潮真正有价值的产出，不是一句“市场太残酷”，也不是一句“以后谨慎”。它应当留下能在下一轮繁荣中执行的规则。

## 把教训带回下一轮旺季

退潮后最容易制定严格纪律，因为痛苦还新鲜；最难的是行情再次顺利时仍然执行。为此，规则要写成可观察动作，而不是情绪承诺：

| 模糊教训 | 可执行规则 |
| --- | --- |
| 以后不贪心 | 单一主线总错误预算达到上限后，不用新代码绕过 |
| 以后少融资 | 主线进入扩散或拥挤阶段，新增盈利优先偿还负债 |
| 以后早点卖 | 原失效条件出现后，在预定时间窗内执行，不等回本 |
| 以后多分散 | 按共同失效原因归类，压力测试同时下跌 |
| 以后保持冷静 | 仓位触发睡眠、冲动交易等预警时，先降到可执行尺寸 |

这些规则仍需在新环境中检验。市场结构会变，过去的精确阈值未必永久有效；但“盈利不能豁免审计”“同因子必须合并计算”“负债会缩短时间”这类机制可以长期保留。

## 退潮不是审判，是压力测试

潮水褪去以后，人最想知道的是：我当初是不是很愚蠢。这个问题几乎没有经营价值。上涨时的你拥有当时的信息、环境和激励；今天的你已经知道结局。用结局审判过去，只会得到一个完美但无法重复的故事。

更有价值的问题是：哪些风险原本就存在，只是在上涨中被收入覆盖？哪条规则因为连续盈利而漂移？哪一部分损失以后仍无法预测，但可以通过规模降低伤害？

退潮不是为了证明过去的人多么失败，而是为了让未来的人少一种失败方式。

> 牛市里检查能力，熊市里检查结构；真正的系统，两种账都要看。
