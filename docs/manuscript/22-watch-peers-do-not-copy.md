# 第二十二章　看同行，不要抄同行

同行提供坐标，不提供答案；表面动作可以复制，底层经营系统不能靠模仿自动获得。

## 两家都卖便宜机票，为什么不是同一种生意

街上两家餐馆都把套餐降到三十元。一家靠单品菜单、稳定供应、快速翻台和低损耗仍能赚钱；另一家菜单复杂、后厨拥挤、采购零散，只是照着价格牌降价。

顾客看到的是同一个价格，老板经营的是两个系统。

投资者看同行也容易停在表面：别人上了什么产品、扩了多少产能、进入哪个概念、用了哪种商业模式。可同行真正有价值的地方，不是提供一份可以照抄的菜单，而是帮助我们看见：共同环境相同以后，什么差异决定了结果。

> 同行是参照，不是答案。

## 比产品以前，先拆经营系统

一家企业的竞争优势很少只来自一个动作。低价背后可能需要：

- 更简单的产品和服务结构；
- 更高的资产利用率；
- 更快的周转；
- 更稳定的供应与组织配合；
- 更低的获客或履约成本；
- 顾客愿意接受的取舍。

如果只复制价格、产品名称或营销口号，却没有复制成本、流程、渠道和组织，模仿者得到的不是同一种商业模式，而是一张更薄的利润表。

研究同行时，至少拆成五层：顾客为什么来、收入从哪里来、成本怎样被压低、关键环节由谁控制、规模扩大以后什么会变坏。

## 西南航空：点对点不是一句口号

西南航空在年报中将其主要网络描述为点对点服务，并与多数美国大型航空公司的枢纽辐射模式区分；年报也讨论了机队、航线结构、运营和客户政策等相互连接的经营要素。[1]

这不意味着点对点天然优于枢纽辐射，也不意味着模仿某一项做法就能复制西南航空的历史结果。航空公司的机场条件、机队、航权、维护、排班、劳资关系、顾客结构和收益管理共同决定单位经济性。系统里的一个动作离开其他条件，含义可能完全改变。

这个案例只说明：

> 商业模式不是一道菜，而是厨房、菜单、动线、采购和顾客预期共同组成的系统。

因此，看到同行成功，不应先问“它做了什么”，而要继续问“这件事依赖哪些条件”“这些条件我们是否具备”“复制以后，优势会不会因拥挤而消失”。

## 产业空间不会平均分给每一家上市公司

一条餐饮街很热闹，不等于每家店都赚钱。利润可能落在拥有稀缺铺位的房东、掌握独家调料的供应商、控制流量的平台，或者翻台最快的少数门店。

产业研究也要把“总蛋糕”和“切蛋糕的刀”分开。终端需求增长只是第一步，股东回报还取决于：

| 变量 | 要问的问题 |
| --- | --- |
| 稀缺性 | 这个环节是否真有供给瓶颈 |
| 议价权 | 成本变化能否转嫁给客户 |
| 技术与良率 | 替代需要多久、付出什么代价 |
| 客户切换 | 更换供应商会不会影响验证、产线或品质 |
| 资本强度 | 增长要吞掉多少新增现金 |
| 竞争格局 | 新进入者会不会迅速压低价格和回报 |

> 赛道规模是顾客一共花多少钱，利润池才是其中多少能留给股东。

## ASML：需求、瓶颈和持续服务是不同层次

ASML的年度报告把系统销售、服务与现场选件、研发投入和客户装机基础分别披露。2025年报告显示，服务与现场选件收入随装机基础、设备使用和升级活动增长；公司也持续投入光刻、计量检测、软件和下一代系统。[2]

这里不能把“产业关键环节”简化成“任何相关公司都值得买”，也不能把ASML的客户关系、技术积累和全球供应体系直接映射成某家A股公司的确定客户、份额或订单。

该案例只提供一个利润池观察方法：设备交付是一笔收入，装机基础可能带来持续服务，技术迭代又要求长期研发和供应链协作。判断价值落点，要同时看产品稀缺、客户生产依赖、后续服务、资本投入与替代难度，而不是只看产业终端需求。

## 大家都做，是需求信号，也是供给警报

同行一致扩产通常有两层含义。

第一层，企业确实看到了需求、价格或政策机会。多个竞争者同时行动，可以增强方向的可信度。

第二层，未来供给正在被培养。今天证明行业好赚钱的事实，可能正吸引明天压低回报的竞争者。

所以，看到“大家都在做”时，需要同时启动两只时钟：

- 需求时钟：订单、终端销量、渗透率和客户资本开支何时兑现；
- 供给时钟：建设、投产、爬坡、良率和销售何时形成真实产出。

高壁垒设备、资源约束和长验证周期可能让供给增长远慢于投资意愿；标准化、容易融资和容易复制的环节，则更容易从共识走向价格战。

## 相对强弱是一场不完美的对照实验

当同行面对同一行业、同一政策或同一市场回调时，差异更有信息量。

可以比较：

- 谁先启动，谁只是后排补涨；
- 谁能承载持续成交，谁只靠小成交拉升；
- 板块分歧时谁更抗跌；
- 利好出现时谁能创新高，谁只反抽；
- 多轮行情中领先者是否稳定。

这像一场不完美的对照实验：共同环境暂时被控制，价格和成交差异帮助我们识别市场正在选择谁。

但相对强弱不能替代产业研究。短期事件、估值位置、市值和流动性都会影响排名；资金核心也不等于长期护城河。正确次序是用同行发现问题，再回到决定性变量和证据链验证。

## 海外映射先映射条件，不映射代码

海外龙头上涨时，最容易出现一种懒惰推理：海外公司受益于某趋势，因此A股名称相近或概念相关的公司也会受益。

真正应该映射的是条件：

1. 海外公司利润来自产业链哪一环；
2. 优势依赖什么技术、客户、资本和组织条件；
3. A股候选是否拥有相同的价值捕获位置；
4. 相关关系处于布局、送样、验证、导入、量产还是收入阶段；
5. 当前价格是否已经把映射预期提前付清。

未经核验的客户、份额、唯一供应和订单关系，只能进入待验证清单，不能进入事实栏。

> 海外核心能证明一条利润路径存在，不能替你证明本地公司已经站在路径上。

## 本章行动卡

### 做什么

- 研究同行时拆解顾客、收入、成本、流程和组织条件。
- 先画产业链，再标注收入、毛利、现金和资本开支的落点。
- 在共同催化和共同回调中比较相对强弱。
- 海外映射先核验条件和证据阶段，再讨论股票。

### 不做什么

- 不因产品相似，就假定商业模式相同。
- 不把行业空间平均分给所有上市公司。
- 不把“大家扩产”只理解成需求确认。
- 不虚构客户、订单、唯一供应商或市场份额。

### 适用条件

适用于半导体、新能源、创新药、消费电子、平台生态等产业链较长、公司角色差异明显的行业，也适用于主线内部核心与补涨的比较。

### 失效条件

若同行的会计口径、地区、产品阶段和商业模式并不可比，停止横向排名，先重建可比口径。重大公司事件也可能暂时扭曲相对强弱。

### 主要风险

最大的风险是把“参照”再次做成答案：要么照抄成功企业的表面动作，要么仅凭短期强弱否定长期竞争力。同行比较的任务是提出更好的问题，不是取消独立研究。
