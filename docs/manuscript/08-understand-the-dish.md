# 第八章　先懂菜，才能看懂生意

行业研究不是给同一张表换几个名词，而是找到这门生意最不能含糊的变量。

## 菜名相同，厨房可能完全不同

两家餐馆都在菜单上写着“招牌牛肉”。一家真正掌握稳定供应、腌制配方和火候，另一家只是从中央厨房进货，再换一个漂亮盘子。顾客第一次看菜单，很难分辨；忙过一轮晚市，成本、复购和出品稳定性就会拉开差距。

股票研究也经常从名字开始：某公司布局人工智能，某产品用于半导体，某企业进入创新药。名字能够指向一个方向，却不能说明公司处在产业链哪一环、跨过了哪一级验证、收入和现金最终能否进入收银台。

如果不理解厨房，投资者只能根据菜单想象利润。故事越新，专业词越多，想象越容易跑在证据前面。

> 不懂菜的人，最容易把菜单上的名词当成厨房里的能力。

## 每个行业都有自己的决定性变量

统一的财务指标能帮助比较公司，却不能替代行业知识。毛利率、收入增长和现金流都重要，但它们背后的因果链在不同行业完全不同。

资源行业的关键可能是成本曲线和供给释放；创新药的关键可能是临床终点、成功概率与专利；半导体设备的关键可能是客户验证、良率、产能和切换成本；消费品牌的关键可能是复购、渠道与定价权。

本书把最能决定一门生意结果、又最不能用通用口号替代的变量，叫作“决定性变量”。它不是永远只有一个，而是要求研究者先回答：如果我只能持续跟踪三项数据，哪三项最能推翻或支持这项投资？

### 锂电与资源制造

不能只看新能源长期需求。还要看资源禀赋、完全成本、现金成本、产能释放、库存、产品价格、资本开支和企业在成本曲线中的位置。需求增长若慢于供给扩张，市场空间可以变大，利润仍可能下降。

### 创新药

不能只看靶点“热门”或市场空间。要看作用机制、临床阶段、终点设计、成功概率、竞争格局、专利期限、授权条款、现金消耗和下一次融资需求。一个科学设想、一次临床试验许可和一款最终上市药物之间，隔着多次可能失败的验证。

### 半导体与高端制造

不能把“国产替代”当作收入确认。要看产品所处工序、客户验证周期、良率、导入进度、量产能力、客户集中度、切换成本和下游资本开支。设备进入客户现场，和形成稳定批量采购，不是同一件事。

> 研究不是知道更多行业名词，而是知道哪一个变量错了，整套故事就要重写。

## 证据有楼梯，故事喜欢坐电梯

产业新闻常使用“布局、送样、验证、导入、量产”等词。它们并非没有价值，但所代表的成熟度不同。

| 阶段 | 它真正说明什么 | 还不能说明什么 |
| --- | --- | --- |
| 布局 | 公司决定投入资源或进入方向 | 技术可用、客户存在、能够盈利 |
| 送样 | 产品具备初步交付形态 | 客户验证通过、形成订单 |
| 验证 | 客户或监管开始检验性能与风险 | 一定导入、一定批量采购 |
| 导入 | 产品进入采购或生产体系 | 规模稳定、毛利可靠 |
| 量产 | 具备批量生产与交付能力 | 销量、售价和利用率足够 |
| 收入 | 产品已经形成会计确认 | 利润高、现金已经收回 |
| 回款 | 销售更接近真实现金 | 资本回报一定优秀 |

本书把事实沿这条楼梯向收银台推进，叫作“证据升级”；把早期表述直接等同于后期结果，叫作“证据跳级”。

> 送样不是收入，验证不是量产，布局不是客户。

每升一级，都需要新的独立证据。公司公告、监管记录、客户采购、收入结构、产能利用率和现金回款，证明的对象各不相同。重复转述同一句公司表态，不会让证据自动升级。

证据越靠后，经营确定性通常越高，但股票赔率不一定越高。市场可能提前定价，等收入完全确认时，预期已经很贵。因此，证据阶梯要与第十八章的隐含预期一起使用：既看事实走到哪里，也看价格已经假定它走到哪里。

## 中际旭创：同一家公司，不同表述在不同台阶

中际旭创2026年5月的投资者关系活动记录提供了一组很清楚的用词边界。公司表示，1.6T产品已经批量出货；3.2T产品仍处于研发完善阶段；NPO方案仍在技术改进和客户验证中。文件同时明确，交流内容不构成业绩预测或承诺。[1]

这三个表述都与下一代光模块有关，却不能获得同样的证据等级。

“批量出货”说明1.6T已经跨过研发和单纯验证，进入商业交付；但仍需继续看销量、价格、毛利、客户集中和回款。“研发完善”说明3.2T方向存在投入，却不能直接写成量产收入。“客户验证”比概念布局更靠前，仍然不等于采购规模已经确定。

公司2024年年报和2026年一季报显示，800G、400G等高端光模块需求已经进入收入和利润，形成了可核验的经营结果。[2][3] 这些成熟产品的兑现，能够提高我们对公司研发与交付能力的理解，却不能自动把3.2T或NPO未来收入提前确认。

这就是行业研究真正需要的精度：相信公司有能力，不等于把每一项在研产品都记入利润；看到早期验证，也不等于否定其价值。把不同产品放回不同台阶，既避免盲目乐观，也避免一概拒绝新技术。

## 创新药：获准试验，为什么还不是获准上市

国家药品监督管理局的药物临床试验审批事项要求申请人提交药学、质量、药理、毒理等研究资料，由药品审评中心进行技术审评，符合要求后获得开展临床试验的许可。[4]

获准开展试验是一项重要证据，意味着项目跨过了前一阶段的资料与安全性审查；它却不意味着药物已经证明有效，更不意味着获得上市许可。后续还要面对临床研究、风险收益评价、生产质量和监管审评等环节。

美国FDA面向患者的药物开发流程也将发现与开发、临床前研究、临床研究、监管审查和上市后监测分成不同阶段。[5] 这份美国资料只能帮助理解阶段结构，不能替代中国药品监管要求。

因此，研究创新药时，必须把“获批临床”“完成某期入组”“达到主要终点”“提交上市申请”“获得批准”和“商业销售”分开。市场可以为远期成功概率定价，但投资者不能把概率语言改写成已经发生的收入。

## Theranos：欺诈不是普通研发失败

证据跳级有时来自过度乐观，有时则涉及故意误导。两者不能混在一起。

美国证券交易委员会2018年指控Theranos及相关高管通过虚假或夸大的技术、经营和财务陈述向投资者募资。SEC在公告中称，公司专有设备实际上只能完成少量测试，大部分检测仍使用传统商用设备。[6]

这个案例可以说明宏大叙事如何压过技术与商业验证，却不能拿来类比每一次研发延期、临床失败或客户验证不及预期。正常创新本来就伴随失败概率；欺诈的关键在于对事实作出虚假或误导性陈述。

投资者需要的是证据纪律，不是把所有早期技术公司都视为骗子。真正的纪律，是允许未知保持未知，允许失败保持失败，也要求管理层的公开说法与可核验事实对应。

## 专业词越多，越要追问“谁付钱”

产业研究容易被技术名词占满。NPO、CPO、先进封装、某种靶点、某种材料路线都可能重要，但一项技术进入上市公司价值，至少要回答五个问题。

1. 它解决了谁的什么问题？
2. 谁拥有关键能力，替代难度在哪里？
3. 客户处在哪一级验证，采购决定由谁作出？
4. 公司为此投入多少资本，量产和良率是否支持利润？
5. 收入、现金和股东回报最终怎样形成？

如果五个问题始终只能回答第一个，研究可能仍停留在科普层。科普帮助理解世界，不自动构成投资证据。

> 技术路线回答能不能做，收银台回答值不值得做。

## 公司说了什么，与事实证明了什么

公开资料也有证据等级。监管公告、审计财报、交易所问询回复、正式业务数据、投资者关系记录、媒体报道和社交平台传闻，能证明的范围不同。

公司在交流中说“积极布局”，只能证明公司表达了方向；公告显示“签署框架协议”，只能证明双方有一项安排；财报出现对应收入，才说明商业化进入会计结果；经营现金流和回款，才能进一步检查收入质量。

“客户不便披露”不能反向证明某个知名客户存在。“唯一供应商”“独家订单”“市场份额第一”等断言，必须有可追溯原始依据。没有依据时，最诚实的写法不是补全故事，而是标注未验证。

## 建一张决定性变量卡

研究一家公司以前，先写下面这张卡。它能迫使信息从“很多”变成“有层级”。

| 栏目 | 必须回答的问题 |
| --- | --- |
| 顾客问题 | 产品解决谁的什么问题？需求是刚性、周期还是政策驱动？ |
| 决定性变量 | 哪三项指标最能决定成败？为什么不是别的指标？ |
| 产业位置 | 公司在哪个工序或利润环节？谁掌握议价权？ |
| 当前证据 | 布局、送样、验证、导入、量产、收入还是回款？ |
| 证据来源 | 公司、客户、监管、财报或第三方？能证明到哪一级？ |
| 竞争与替代 | 对手是谁？客户切换成本和技术替代风险是什么？ |
| 资本要求 | 还要投入多少资金？良率、产能利用和现金消耗如何？ |
| 失效条件 | 哪项事实出现，必须下调证据等级或停止研究？ |

这张卡不要求散户变成工程师、医生或化学家。它要求的是知道自己不懂什么，并把最重要的未知留在纸面上。专业研究的起点不是无所不知，而是不让未知偷偷变成确定。

## 做什么

为每个行业寻找自己的决定性变量。把公司表述贴到证据阶梯，区分事实、判断和未验证。每次研究更新都写清“新增了什么证据、升了哪一级、还缺什么”。

用公司财报、交易所和监管资料核验关键事实；媒体与社交平台可以提供线索，不能单独完成确认。将产品进展继续连接到收入、利润、现金和资本回报。

## 不做什么

不把行业空间直接写成公司利润。不把布局等同于客户，不把送样等同于订单，不把验证等同于量产。不虚构供应关系、唯一性和市场份额。不因为技术听起来先进，就省略估值与竞争。

## 适用条件、失效条件与主要风险

证据升级框架特别适合研发、制造、医药和长验证周期行业。软件订阅、平台业务和消费品的验证链可能更短，不能机械套用制造业阶段；它们仍需要自己的用户、留存、付费和现金证据。

公开披露可能滞后，商业机密也会限制信息完整性。证据不足不等于事实一定不存在，只意味着账户不能按已经存在来分配同样仓位。最主要的风险，是形式上填写了阶梯，实际上仍用同一来源反复证明自己。

> 研究的目标不是把故事讲圆，而是知道故事在哪一级、下一步可能在哪里断。
