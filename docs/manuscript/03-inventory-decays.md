# 第三章　库存会腐败，仓位不是静态资产

真正腐败的不是股票，而是继续持有它的理由。

## 昨天的资产，今天可能开始发酸

餐馆老板清晨买回一批牛肉。刚入库时，它是资产：能做成菜，卖给顾客，换回现金。可冰箱温度突然升高，客流又比预期少。第二天，牛肉边缘开始变色；第三天，已经有了异味。

老板仍然可以拿出采购单，说每斤买得很便宜，也可以强调供应商多么可靠。可采购价无法延长保质期，过去的判断也不能阻止库存变质。

股票当然不会像牛肉一样物理腐烂。上市公司也可能持续经营很多年。本章所说的“库存腐败”，指的是账户中继续持有一只股票的原始理由，随着证据、趋势、流动性和时间变化，已经不再支持原来的交易身份和仓位。

一只股票买入后，并没有成为静止财产。它仍在消耗资本、注意力和机会。理由推进，它可能从观察仓升级为核心；理由停滞，它的保质期会缩短；理由失效却被新故事替代，它就开始变成烂库存。

> 买入价记录你从哪里来，不能决定你还该不该留下。

## 股票没有保质期，交易理由有

不同交易的时间结构不同。突破交易买的是突破后的延续，分歧低吸买的是关键位置的承接，主线交易买的是资金和产业共振，长期投资买的是多年现金创造能力。它们的验证窗口不能混在一起。

因此，保质期不是统一的三天、十天或一个季度，而是“原始理由应该在多长时间内看到什么证据”。

| 交易身份 | 原始理由 | 应出现的证据 | 常见失效表现 |
| --- | --- | --- | --- |
| 突破交易 | 价格越过平台，增量资金接力 | 1—3日守住平台、相对强度延续 | 迅速跌回平台且反抽无力 |
| 主线轮动 | 方向重新获得资金偏好 | 板块核心共振、成交保持 | 只有后排脉冲，核心持续弱化 |
| 事件交易 | 公告或催化改变短期预期 | 市场正反馈、后续事实推进 | 利好钝化、事件被证伪或兑现即退潮 |
| 中长期投资 | 竞争力与现金创造长期改善 | 经营指标按阶段兑现 | 证据链断裂、资本回报恶化、估值失去赔率 |

没有写清交易身份，任何持仓都可以被无限延期。短线失败后，可以说行业长期空间大；长期逻辑遇到价格上涨，又可以改口说顺势交易。时间尺度不断迁移，投资者永远不必承认原来的理由已经过期。

本书把这种行为叫作“失败改名”。它不是正常地更新观点，而是在亏损以后，用一套验证周期更长、更难证伪的新语言，保护旧仓位免于重新审查。

> 交易身份可以改变，但改变身份等于发生了一笔新交易。

新交易应重新经过研究、估值、仓位和退出检查，不能因为股票已经躺在账户里，就免除这些程序。

## 库存腐败的五步链条

烂库存通常不是突然形成，而是沿着一条熟悉的路径慢慢变质。

### 第一步：原始理由很清楚

买入时也许是平台突破、板块回流、订单验证或估值修复。交易有身份，也有一项可以观察的关键证据。

### 第二步：证据开始变弱

突破没有延续，板块强而个股弱，验证迟迟不推进，或者经营数据与预期出现偏差。此时问题往往仍然可控，仓位也可以被重新调整。

### 第三步：解释开始扩张

投资者不再检验原始理由，而是寻找更远、更大的故事：短期不行就看明年，产品没有放量就谈市场空间，个股弱就说资金不懂价值。

### 第四步：交易身份被改名

原本三天内应验证的突破交易，变成准备持有三年的成长投资；原本小仓试错的事件交易，变成不能卖的核心资产。

### 第五步：账户被旧决定占用

仓位无法释放，新机会出现时没有现金；投资者的研究时间被一只失效股票不断吸走；亏损越大，越难承认最初只是一笔普通交易。

这条链上，最危险的不是第一步判断错误，而是第四步拒绝让错误结算。小错误本来只需要付出小成本，身份改名却把它变成长期资本占用。

## 成本价是历史，不是顾客

餐馆顾客不会因为老板进货贵，就愿意为一道菜多付钱。股票市场也不会因为你的成本价更高，就给你一条回本路线。

成本价当然有记账意义。它帮助计算盈亏、税费和执行质量，也会影响心理。但在判断是否继续持有时，更有用的问题是：如果今天账户里没有这只股票，我是否愿意以当前价格、当前证据和当前仓位重新买入？

如果答案是否定的，继续持有就需要一个独立理由，而不能只剩“已经亏了这么多”。过去投入的资金是沉没成本，未来的资本仍然可以重新选择。

“跌多了”也不是理由已经恢复的证据。价格下降可能改善赔率，却也可能反映基本面、主线地位或流动性进一步恶化。只有当新价格与新证据共同重建吸引力，低价才成为新的买入理由。

## Target：承认库存错了，代价也不会消失

2022年，Target面对消费结构变化和过剩库存，宣布采取追加降价、清理多余库存、取消订单并调整品类计划等措施。公司同时明确，这些行动会压低当季利润率，但希望通过处理库存改善后续经营。[1]

这个案例最重要的地方，不是Target后来表现如何，而是管理层没有因为库存已经买入，就假定它仍然具有原来的价值。需求变化以后，过去合理的备货也会变成占用现金、仓储和折扣空间的负担。处理库存会确认损失，却可能阻止问题继续扩大。

卖出一只失效股票也类似。损失不是卖出动作创造的；卖出只是让账户停止继续承担原来的风险。真正的损失往往在理由失效、价格下跌时已经发生。拒绝处理，只是把“已发生的损失”和“未来仍可能发生的损失”绑在一起。

Target不能证明所有库存都应快速清理。零售企业需要为未来销售备货，投资者也需要给正确假设合理时间。关键是，当真实需求已经改变，库存的处理标准也必须改变。

> 确认损失会让报表难看，拒绝确认可能让生意更难看。

## Zillow Offers：小误差进入大库存

Zillow Offers把房屋估值、买入、翻修和转售连接成规模化业务。2021年11月，Zillow决定关闭该业务。公司向美国证券交易委员会提交的文件提到房价预测的不确定性、运营容量限制和执行挑战，并确认约3.044亿美元库存减记，因为部分房屋的预期售价低于买入成本及后续费用。[2]

这不是“算法算错一次”的简单故事。一项判断进入大规模库存以后，会同时受到定价、融资、施工、周转和市场流动性的影响。预测误差本来可能很小，规模和执行链却会把它放大。

在账户中，类似机制发生在持仓尺寸上。同一个错误放在观察仓里，可能只是一次可学习的试错；放在高度集中、流动性较差的仓位里，就会迅速侵蚀经营权。仓位怎样反推错误预算，留到第九章完整讨论。这里先记住：库存的性质与规模不能分开，但库存是否变质，首先由理由和证据决定。

序章已经用Zillow说明观点如何进入资产、负债、库存和现金；本章只调用其中一层：判断一旦变成库存，就不再只受判断本身控制。

## 三种“库存”不要混为一谈

投资研究里常同时出现公司库存、行业库存和账户库存。它们使用同一个词，却不是同一对象。

### 公司库存

指上市公司资产负债表中的原材料、在产品和产成品。它影响周转、减值、毛利和现金占用。

### 行业库存

指产业链上下游共同持有的库存。它影响供需、价格和补库周期，常决定一个行业处于去库、补库还是过剩阶段。

### 账户库存

指投资者持有的头寸，以及这些头寸占用的现金、风险容量和注意力。它的“变质”不是会计减值，而是原始持有理由与当前事实不再匹配。

公司库存改善，可能支持股价逻辑，却不能自动证明账户仓位合理；行业进入补库，可能提升景气，却不能取消估值和市场拥挤；账户持仓下跌，也不能直接推出公司库存恶化。先分清对象，隐喻才不会替代专业判断。

## 价格下跌不等于库存腐败

如果把每一次下跌都叫作腐败，长期投资和趋势中的正常波动都无法存在。判断理由是否变质，至少要看四类证据。

1. **事实证据**：产品、客户、订单、利润和现金流是否按原假设推进。
2. **市场证据**：价格结构、相对强度和利好反应是否持续弱化。
3. **时间证据**：原本应在什么窗口看到验证，当前延迟是否仍有合理解释。
4. **账户证据**：仓位、融资和流动性是否允许继续验证，而不夺走经营权。

价格下跌但事实推进、板块核心同步调整、账户有充足等待权，可能只是波动。价格还没大跌，但证据停滞、同行已替代、利好钝化，也可能已经开始腐败。

所以，腐败不是颜色，而是关系：原始理由与新事实之间的关系。

## 机会成本是冰箱占用费

一批长期卖不动的食材，不只可能降价，还占着冰箱位置。新鲜食材来了，老板可能因为没有空间而无法进货。

持仓也会占用未承诺资本。即使一只股票价格没有继续下跌，只要它长期停滞、理由没有推进，就在消耗机会成本。对高仓位账户而言，机会成本尤其真实：新方向出现时，只能先卖旧仓；若旧仓流动性差或正在亏损，投资者便容易为了避免痛苦而错过重新配置。

机会成本不能被滥用成频繁追逐最强板块的借口。任何账户都不可能覆盖每次上涨。只有当替代机会的持续性和相对优势，足以覆盖换仓成本、追高风险与原持仓剩余赔率时，它才构成可行动的证据。

## 给每笔持仓贴一张库存标签

买入以后，不要只记录成本和盈亏。为持仓保留一张标签，至少写清以下内容。

| 项目 | 必须回答的问题 |
| --- | --- |
| 交易身份 | 这是突破、主线轮动、事件交易，还是中长期投资？ |
| 原始理由 | 当初最关键的三条证据是什么？哪一条最重要？ |
| 理由保质期 | 在什么时间窗口内，应当看见什么进展？ |
| 失效条件 | 哪些事实、市场或账户变化意味着原身份不再成立？ |
| 当前证据 | 新证据是在升级、停滞、降级，还是彼此矛盾？ |
| 无仓检验 | 如果今天没有持仓，是否愿意以同样规模重新买入？ |
| 当前行动 | 继续持有、降低承诺、退出观察，还是按新身份重新立项？ |

“按新身份重新立项”很重要。公司可能在短线逻辑失效后，确实出现新的长期投资价值；一只长期持仓也可能因市场进入主升阶段而增加交易仓。但新身份必须有新证据、新估值、新仓位和新退出，不能只改备注名称。

## 处理烂库存，不等于永远离开

卖出不是永久判决。理由失效时退出，未来证据重新成立后仍可以再买。真正阻碍重新进入的，常常不是市场，而是投资者把卖出理解成羞耻，于是要么死扛，要么卖出后发誓永不回头。

更好的机制是预先写下重新进入条件：关键证据恢复、价格重新站回结构、板块地位得到确认、账户风险容量恢复。这样，退出只是结束旧交易，不是关闭对这家公司的研究。

这也是库存管理与价值判断的根本区别。老板可以暂时停止采购一道菜的原料，等顾客回来、供应恢复或成本下降后再上菜单。停止备货不等于否定这道菜永远没有价值。

> 退出旧理由，是为了给新证据留下位置。

## 不要用“长期”给冰箱断电

长期投资需要耐心，但耐心不是取消保质期。真正的长期持有，依赖持续的竞争优势、资本回报和现金创造；它需要更长的证据窗口，却不是没有证据窗口。

当原始理由失效时，最危险的不是亏损本身，而是研究开始只为持仓辩护。人会主动寻找支持材料，回避反证，用“长期”把所有延迟合理化。于是，账户里留下的已经不是一个经过验证的长期资产，而是一笔拒绝结算的旧交易。

股票不会因为你喜欢它就停止变化，交易理由也不会因为写上“长期”就自动保鲜。定期打开冰箱，不是缺乏信仰；恰恰因为资本有限，才必须知道里面还有什么能够卖给顾客。

> 好库存等待需求，烂库存等待回本。
