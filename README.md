# 投资经营学

> 让复杂的投资问题，可以用普通人的经营直觉来思考。

《投资经营学》是一套面向普通投资者的开放思想体系。它不试图把金融知识“讲得浅一点”，而是把抽象概念还原成人人熟悉的生活与经营经验：客流、菜单、库存、房租、现金、扩店和关店。

我们采用一条固定的认知路径：

**生活画面 → 经营含义 → 金融概念 → 投资行动**

例如：

- 仓位不是表达信心，而是表达愿意承担多少错误。
- 市场什么时候纠错不知道，融资什么时候到期写在合同里。
- 真正应该分散的不是股票，而是犯错的原因。
- 退出不是给过去定罪，而是把未来重新交给自己。

## 明确不公开的内容

本仓库不会收录任何个人投资账户材料，包括但不限于：

- 真实持仓、成交记录和盈亏数据
- 券商截图、账户号码和资金规模
- 能够反推出个人财富或交易身份的信息
- 私人对话、未公开研究笔记和第三方受限材料

参与贡献时也请不要提交上述内容。需要举例时，请使用公开资料或明确标注为虚构、标准化的数据。

## 这不是什么

本项目不是荐股服务，不提供实时买卖信号、收益承诺或个股目标价。文中案例用于解释和检验方法，不代表当前观点，也不构成投资建议。投资有风险，任何决策都应由读者独立作出并自行承担结果。
