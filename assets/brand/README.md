# 《投资经营学》视觉识别 v0.1

## 核心意象

视觉不使用牛熊、K线、金币、火箭或西装精英，而以一间正在认真经营的小餐馆为核心：

- 门外的顾客代表市场需求；
- 菜单代表产品与投资方法；
- 食材和货架代表库存与仓位；
- 账本和收银台代表利润、现金流与复盘；
- 灯光代表仍然拥有经营权和下一次机会。

## 使用原则

- 标识四周至少保留其宽度四分之一的留白；
- 不改变色彩关系，不增加渐变、阴影和金融符号；
- 对外文案优先使用“投资经营学”和“像经营一家餐馆一样思考股票”；
- 图片生成部分用于本项目内容传播，沿用仓库内容许可证 CC BY-NC-SA 4.0。
